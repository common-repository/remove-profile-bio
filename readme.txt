== Description ==

The Remove Profile Bio plugin allows you to quickly remove the biographical information field from the edit profile page on your site.

Features:

* Simple install.  Activate, and the field disappears.
* Renames the section header from "About Yourself" to "Password"
* Does not use jQuery to hide the field.


== Installation ==

1. Upload `remove-profile-bio.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
